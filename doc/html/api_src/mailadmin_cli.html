mailadmin_cli module¶

Command-line interface for administering the mail server database.

A simple command-line interface for inspecting and manipulating the database associated with a Postfix- and Dovecot-based mail server configured according to the instructions from Linode (https://www.linode.com/docs/guides/email-with-postfix-dovecot-and-mysql/).
